roBrowserLegacy

Open Source Ragnarok Online Web Client

Author: Vincent Thibault
Site: robrowser.com
Source: github.com/MrAntares/roBrowserLegacy

roBrowser is an open source project based on the game Ragnarok Online.
It's not affiliated in any way with Gravity.

The concept is to reproduce the game using web technologies (HTML5, Javascript, WebGL) to bring it to Web Browsers.

Cross-platform: runs on Windows, Linux, macOS, and any device with WebGL support.
