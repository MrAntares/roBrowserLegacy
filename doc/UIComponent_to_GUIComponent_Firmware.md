# L2 Compressed reflex matrix

# Firmware Rules

1. Query inside shadow.
2. Geometry belongs to host.
3. Bind once.
4. Restore on append.
5. Cleanup on remove.
6. Visibility is explicit.
7. Events are explicit.
8. Formatting is preserved.
9. Input fields are protected.
10. Async DOM writes are guarded.
11. Shadow boundaries invalidate global assumptions.
12. Re-test every interaction path after migration.
